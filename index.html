Oipc ground station DEB repository

  • Project repo
  • Bookworm DEB
  • Bullseye DEB

Import GPG key

wget -qO - https://seriyps.github.io/oipc_gs/public.gpg | sudo gpg -o /usr/share/keyrings/oipc_gs.gpg

Add repository to bookworm

echo "deb [signed-by=/usr/share/keyrings/oipc_gs.gpg] https://seriyps.github.io/oipc_gs/oipc-bookworm bookworm main" | sudo tee /etc/apt/sources.list.d/oipc_gs.list

Add repository to bullseye

echo "deb [signed-by=/usr/share/keyrings/oipc_gs.gpg] https://seriyps.github.io/oipc_gs/oipc-bullseye bullseye main" | sudo tee /etc/apt/sources.list.d/oipc_gs.list

Install Oipc ground station


    sudo apt update
    sudo apt install openipc-gs-radxa3-basic
