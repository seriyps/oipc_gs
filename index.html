Oipc ground station DEB repository

  • Project repo

Import GPG key

curl -s https://seriyps.github.io/oipc_gs/public.gpg | sudo gpg --dearmor -o /usr/share/keyrings/oipc_gs.gpg

Add repository to bookworm

echo "deb [signed-by=/usr/share/keyrings/oipc_gs.gpg] https://seriyps.github.io/oipc_gs/oipc-bookworm bookworm main" | sudo tee /etc/apt/sources.list.d/oipc_gs.list

Add repository to bullseye

echo "deb [signed-by=/usr/share/keyrings/oipc_gs.gpg] https://seriyps.github.io/oipc_gs/oipc-bullseye bullseye main" | sudo tee /etc/apt/sources.list.d/oipc_gs.list

Add WFB-NG repository

See https://apt.wfb-ng.org/


curl -s https://apt.wfb-ng.org/public.asc | sudo gpg --dearmor --yes -o /usr/share/keyrings/wfb-ng.gpg
echo "deb [signed-by=/usr/share/keyrings/wfb-ng.gpg] https://apt.wfb-ng.org/ $(lsb_release -cs) release-25.01" | sudo tee /etc/apt/sources.list.d/wfb-ng.list

Install Oipc ground station


    sudo apt update
    sudo apt install openipc-gs-radxa3-basic

Available packages

  • python3-adaptive-link-gs - Adaptive link ground station daemon
  • msposd - Msp OSD ground-station rendering daemon
  • pixelpilot-rk - PixelPilot for Rockchip-based boards
  • rtl8733bu-dkms - patched RTL8733BU DKMS WiFi driver
  • rtl8812au-dkms - patched RTL8812AU DKMS WiFi driver
  • rtl8812eu-dkms - patched RTL8812EU DKMS WiFi driver
  • linux-ina2xx-module - INA2XX kernel module (built for specific kernel version)
